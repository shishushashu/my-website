# Incredible counter
A Website to count things like your score.
## Features
- **Functional counter** with the plus and minus buttons
- A hidden **easter egg** if you click on the counter number
- **A quote** to make it more interasting

## Technology
- HTML5
- CSS3
- JavaScript(Vanilla)

## How to use
1. Open the index.html
2. Use the buttons to count your score
